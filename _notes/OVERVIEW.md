# 百韻管樂團網站系統 — 功能總覽

> 本文件說明系統的功能範圍與操作流程，適合向團員、幹部或非技術背景的人介紹。
> 最後更新：2026-07-11

---

## 為什麼要做這個系統？

原本的百韻網站是靜態頁面，只能顯示固定內容，無法做到：
- 區分「誰看得到什麼」（一般民眾 / 團員 / 幹部）
- 即時追蹤排練出席、請假、借用狀況
- 儲存財務紀錄、通訊錄等敏感資料

新系統透過帳號登入確認身份，不同角色看到的內容與功能不同。

---

## 三種使用身份

| 身份 | 說明 |
|------|------|
| **一般民眾** | 不需登入，只能看公開頁面與公告 |
| **團員** | 登入後可查看活動、申請請假、瀏覽樂譜 |
| **幹部** | 登入後另有所有管理功能 |

---

## 功能一覽

### 不需登入就能使用

| 功能 | 說明 |
|------|------|
| 首頁 / 關於 / 章程 | 原 Hugo 網站的公開頁面，搬入新系統 |
| 公開公告 | 對外公布的活動資訊、最新消息 |
| 校友報到申請 | 校友填寫申請表，系統記錄後等待幹部審核 |
| 申請狀態查詢 | 用 Email 查詢自己的申請進度，不需帳號 |

### 團員登入後可以做什麼

| 功能 | 說明 |
|------|------|
| 首頁 | 顯示下次排練資訊、待審假單提醒 |
| 公告 | 公開公告＋團員限定公告 |
| 演出活動 | 查看籌備中與已結束的演出、各場排練詳情與進度記錄 |
| 申請請假 | 針對特定場排練提出請假，填原因送出 |
| 我的請假紀錄 | 查看歷史申請的審核狀態 |
| 演出分譜下載 | 收到通知後登入，依自己的樂器下載該演出的分譜 |
| 樂譜庫存 | 瀏覽、搜尋、下載樂譜 PDF |

### 幹部另有的管理功能

| 功能 | 說明 |
|------|------|
| 公告管理 | 新增 / 編輯 / 刪除 / 發布公告；草稿不對外顯示 |
| 演出活動管理 | 建立演出、新增排練、管理演出曲目順序；曲目確定後推播 LINE 通知 |
| QR Code 簽到管理 | 每場排練產生專屬 QR Code，控制有效時間與啟用狀態 |
| 校友報到審核 | 查詢/篩選所有申請、核准或拒絕、拒絕後可重新開放審核、手動新增/編輯/刪除申請紀錄；核准後自動建立帳號並將帳密以 Email 寄送給本人 |
| 請假審核 | 核准或拒絕請假申請，核准後自動更新出席紀錄 |
| 團員通訊錄 | 按樂器分組，電話與 Email 幹部限定可查；可搜尋/篩選在團或已退團；可新增團員（帳密以 Email 寄送）、編輯資料、標記退團／恢復在團、無相關紀錄的帳號可直接刪除（管理員可強制刪除已有相關紀錄的帳號）|
| 財務管理 | 收入 / 支出記錄 |
| 會費繳納狀況 | 按期別顯示全體團員繳費狀態（已繳 / 未繳 / 無紀錄） |
| 樂譜庫存管理 | 新增／編輯樂譜，按樂器族群批次上傳分譜，管理對外交換紀錄 |
| 公用財產管理 | 樂器、譜架、音響等公用物品的借用管理 |
| 財產借用現況 | 顯示所有借出中的物品，逾期者自動標記警示 |
| 場地管理 | 管理常用排練 / 演出場地與費用時段 |
| 報表列印 | 排練出席、財產借用現況、會費繳納、請假統計，均可列印 |

## 各系統操作流程

### 校友報到

新校友不需帳號就能申請加入，幹部審核後系統才建立正式帳號。

```
校友在公開頁面填寫報到申請表（姓名、樂器、畢業年份、聯絡方式）
        ↓
校友用 Email 查詢申請進度，不需等電話通知
        ↓
幹部進入審核頁，核准或拒絕
        ↓
核准 → 系統自動建立帳號，帳號與臨時密碼以 Email 寄送給本人
拒絕 → 申請標記為已拒絕
        ↓
校友用臨時密碼登入 → 系統要求先設定自己的新密碼 → 完成後正常使用
```

---

### 幹部手動新增團員

適合指導老師、或不透過網路報到的人，由幹部直接幫忙建檔，不需要走校友報到申請表。

```
幹部進入團員通訊錄 → 點「新增團員」
        ↓
填寫基本資料（姓名、Email、樂器、聲部、畢業年份、電話）
        ↓
系統自動建立帳號，帳號與臨時密碼以 Email 寄送給本人
        ↓
本人用臨時密碼登入 → 系統要求先設定自己的新密碼 → 完成後正常使用
```

> 跟校友報到走的是同一套「臨時密碼 + 登入後強制改密碼」機制，差別只在資料從哪裡來
> （校友自己填申請表，或幹部直接幫忙建檔），角色固定都是團員。

---

### 演出活動與排練

從籌備到結束，所有活動資訊與排練記錄都集中在這裡。

```
幹部建立演出活動（名稱、類型、預定演出日期）
        ↓
逐一新增每次排練（日期、場地）
        ↓
設定演出曲目順序
        ↓
團員在活動列表查看演出資訊與各場排練詳情
        ↓
每次排練結束後，幹部填寫排練摘要（進度、待改進事項、下次重點）
        ↓
演出結束，活動標記為已結束
```

---

### QR Code 簽到

每場排練產生專屬 QR Code，團員掃碼即完成簽到，不需安裝任何 APP。

```
幹部產生當天排練的專屬 QR Code，設定有效時間
        ↓
用手機螢幕顯示，或列印貼在排練室門口
        ↓
團員用手機鏡頭掃描 → 網頁自動開啟 → 點「簽到」完成
        ↓
幹部後台即時看到出席名單
        ↓
（可隨時手動停用舊 QR Code，使其失效）
```

---

### 請假申請

團員事先提出請假，幹部審核，核准後出席紀錄自動更新。

```
團員進入排練詳情頁，或直接在演出活動詳情頁的排練列表 → 點「請假」，填寫原因送出
        ↓
幹部進入審核頁，看到待審申請，核准或拒絕
        ↓
核准 → 出席紀錄自動標記為「請假」
拒絕 → 出席紀錄不異動
        ↓
團員在「我的請假紀錄」查看審核結果
```

> 若團員已先 QR 簽到再補交請假，幹部核准後出席紀錄**不會**被覆蓋，仍保留為「出席」。

---

### 公告

公告分三種可見範圍，支援草稿，確認內容後再發布。

```
幹部新增公告，填寫標題與內容
選擇可見範圍：公開 / 團員限定 / 幹部限定
        ↓
存為草稿，可繼續修改，對外不顯示
        ↓
確認後點「發布」
        ↓
公告對外出現，各人依身份自動篩選，只看得到有權限的公告
```

---

### 樂譜庫存

所有樂譜統一管理，團員可自行搜尋與下載，不需向幹部索取。

```
幹部新增總譜，上傳 PDF，記錄來源（購買 / 交換 / 捐贈）
        ↓
幹部為該曲目批次上傳分譜（按豎笛、薩克斯風等樂器族群分組）
        ↓
團員依類型（總譜 / 分譜）、樂器或曲名關鍵字搜尋
        ↓
點入曲目詳情 → 下載 PDF
（無 PDF 的曲目顯示提示：僅有實體紙本）
```

---

### 公用財產借用

樂器、譜架、音響等公用物品統一管理，幹部可即時掌握借用狀況。

```
幹部登記借出記錄（借用人、預計歸還日期）
        ↓
借用人使用財產
        ↓
物品歸還後幹部登記歸還日期 → 從借用清單消失
        ↓
「財產借用現況」報表顯示所有借出中物品
逾期未還者自動標記警示
```

---

### 財務與會費

所有收支記錄與會費繳納狀況集中管理，幹部可按期別查看報表。

```
【收支記錄】
幹部登記每筆收入或支出，可上傳收據、關聯演出活動

【會費追蹤】
幹部建立該期別（如「2026 上半年」）每位團員的應繳記錄
        ↓
收款後登記繳費日期與收款幹部
        ↓
「會費繳納狀況」報表依期別顯示全體已繳 / 未繳 / 無紀錄名單
```

---

### 演出分譜

幹部確定演出曲目後，系統推播 LINE 通知，團員登入即可依自己的樂器下載分譜。

```
幹部確定演出曲目，將分譜上傳至系統（依樂器分類）
        ↓
LINE 群組推播：「XX 演出曲目已確定，請登入網站下載分譜」
        ↓
團員登入 → 進入該演出頁面
        ↓
系統依登入者的樂器，列出對應的分譜
  → 若同樂器只有一份檔案（聲部合併）：直接顯示下載按鈕
  → 若同樂器有多份檔案（各聲部獨立）：列出所有聲部讓團員自行選擇
        ↓
團員下載所需的分譜
```

> 聲部安排可能因指揮調度而異動，系統不強制指定每位團員的聲部，由團員自行判斷下載哪份。

## 未來方向

若系統成熟，計畫開放給其他管樂團使用。
每個樂團使用自己的子網域（如 `fjcwo.system.com`），資料完全隔離，互不干擾。
